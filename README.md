# MySaju (마이사주) 개인정보 처리방침

**시행일: 2026년 2월 24일**

## 1. 수집하는 개인정보

MySaju는 서비스 제공을 위해 다음 정보를 수집합니다.

- 이름: Google 또는 Apple 로그인 시 제공되는 이름
- 이메일 주소: Google 또는 Apple 로그인 시 제공되는 이메일
- 생년월일시: 사주팔자 분석을 위해 사용자가 직접 입력
- 구매 내역: 구독 상태 확인을 위한 인앱 구매 정보

## 2. 개인정보의 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다.

- 사주팔자 분석 및 운세 서비스 제공
- AI 채팅 상담 서비스 제공
- 사용자 계정 관리 및 인증
- 구독 결제 처리 및 관리

## 3. 개인정보의 보유 및 파기

- 회원 탈퇴 시 모든 개인정보는 즉시 삭제됩니다.
- 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관합니다.

## 4. 개인정보의 제3자 제공

MySaju는 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외입니다.

- 법령에 의한 요청이 있는 경우
- 서비스 제공을 위해 필요한 최소한의 정보를 위탁하는 경우 (Supabase 데이터베이스, Google/Apple 인증)

## 5. 개인정보의 안전성 확보 조치

- 모든 데이터는 암호화된 통신(HTTPS)을 통해 전송됩니다.
- 사용자 인증은 Google 및 Apple의 보안 인증 시스템을 사용합니다.
- 데이터베이스 접근은 인증된 요청에 한해 허용됩니다.

## 6. 이용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

- 개인정보 열람 요청
- 개인정보 수정 요청
- 회원 탈퇴 및 개인정보 삭제 요청
